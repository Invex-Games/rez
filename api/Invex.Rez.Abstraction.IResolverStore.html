AddFunction(string, Func<FunctionCall, string>)

Adds a function to the store.
If a function with the same name already exists, it is replaced.

void AddFunction(string name, Func<FunctionCall, string> function)

Parameters

name string

The name of the function to add.

function Func<FunctionCall, string>

The function to add.
Note: the function should never return null; instead, return an empty string for a blank result.

Remarks

The braces, parentheses, and arguments that denote a function in a template are not included in the name - for the template {repeat(abc,3)}, the name is "repeat".
