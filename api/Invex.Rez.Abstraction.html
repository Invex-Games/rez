Namespace Invex.Rez.Abstraction

Structs

FunctionCall

Contains the arguments for a Rez function call.

When an IResolver encounters a function in a Rez template (e.g., {name(args)}), it constructs a FunctionCall from the text between the parentheses and passes it to the function delegate obtained from an IResolverSource.

Interfaces

IResolver

Provides functionality for resolving variables and functions in a Rez template into an output string.

IResolverSource

Provides a map of variables and functions to be used by an IResolver.

IResolverStore

Provides a map of variables and functions to be used by an IResolver, as well as the ability to add and remove variables and functions at any time.
