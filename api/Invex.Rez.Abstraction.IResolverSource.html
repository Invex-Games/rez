ResolveFunction(string)

Resolves a function with the given name.

Func<FunctionCall, string>? ResolveFunction(string name)

Parameters

name string

The name of the function to resolve.

Returns

Func<FunctionCall, string>

A delegate that produces the function's result from a FunctionCall, or null if this source cannot resolve the function.
Returning null allows the next source registered with the IResolver to be queried.

Remarks

The braces, parentheses, and arguments that denote a function in a template are not included in the name - for the template {repeat(abc,3)}, the name is "repeat".
